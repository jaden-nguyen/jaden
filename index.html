Type Racer

This simple program prints a discrete collection of words in which the user can type to measure typing speed.

Weather App

Word Counter

This application takes in live microphone input and increments a counter for specific keywords defined by the user.
It displays the count of each word in a separate window to be used in conjunction with different applications. The keywords
and the display words/windows are customizable. Built core functionality of recognizing speech using open-source
CMUSphinx and SpeechRecognizer library. Implemented optimized user interface with Python's Tkinter.
